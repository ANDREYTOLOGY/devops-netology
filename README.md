## Игнорируемые файлы

Благодаря файлам .gitignore, следующие типы файлов не будут попадать в репозиторий:

### В основном каталоге проекта:
- Служебные файлы операционных систем (например, .DS_Store для macOS)
- Файлы с чувствительными данными (пароли, ключи API)
- Временные и кэшированные файлы

### В каталоге Terraform/:
- Файлы состояния Terraform (*.tfstate) - они содержат конфиденциальные данные
- Кэш плагинов и модулей (.terraform/) - их можно восстановить командой terraform init
- Файлы переменных с секретами (*.tfvars) 
- Логи аварийных завершений (crash.log)
